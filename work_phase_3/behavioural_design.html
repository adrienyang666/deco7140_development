Button Styles

This section demonstrates how buttons on the site communicate interaction and support user confidence through visual feedback.

Primary Button

This button uses strong colour contrast and clear wording. It is the main call-to-action. Visual states (hover, active) provide immediate feedback.

Secondary Button

This secondary button is visually lighter but still noticeable. It is used for supporting actions and maintains affordance through shadow and border changes.

Disabled Button

This disabled button is visibly inactive and not interactive, providing a clear signifier of unavailable actions.

Icon-only Button

This icon-only button includes an ARIA label to ensure accessibility for screen readers. It is used in compact UI areas like navigation bars.
