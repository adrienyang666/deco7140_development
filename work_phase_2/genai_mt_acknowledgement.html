GenAI and MT Acknowledgement

Context

This assessment involved creating a multi-page user research and analysis website for the Food and Culinary Communities project. I used ChatGPT to assist in generating interview prompts and responses, summarising user insights into user stories, and producing HTML and CSS snippets. Grammarly was also used to refine the grammar of written reflections.

GenAI Tool Usage Breakdown

  • User Research: Used ChatGPT to simulate realistic interviews with diverse personas.
  • User Story Development: Generated and refined user stories from interview responses using AI.
  • HTML & CSS Code: ChatGPT assisted with HTML layout structure, flip cards, navigation, and responsive helper classes.
  • Design Interactions: Supported hover/focus accessibility behavior and interactive highlights for user stories.
  • Writing & Editing: Grammarly checked grammar and readability. ChatGPT refined reflection and instruction-following text.
  • Prompt Iteration: Prompts were tested and refined to avoid biased or overly generic results and improve relevance.

Checklist Application

Rules and Boundaries

I checked the assessment task sheet and the ECP to confirm that the use of GenAI tools was permitted for user research and design support. I ensured all generated content was critically reviewed and modified.

Tool Selection

I chose ChatGPT for its conversational capabilities and idea generation, and Grammarly for proofreading. These tools supported my design thinking while allowing space for personal input and learning.

Privacy Considerations

I did not include any personal or sensitive data in my prompts. Personas used were fictional and AI-generated.

Reviewing and Verifying Content

I reviewed all generated outputs to ensure accuracy and coherence. HTML was tested and edited, while user stories were revised for clarity and specificity.

Ethical Considerations

I ensured all text was inclusive and free of cultural or gender bias. Prompts were refined to avoid stereotypical outputs.

Acknowledgment and Attribution

This page documents my use of AI tools. Comments were added in code where relevant to indicate AI-supported sections.

Reflection

Using GenAI tools helped accelerate the ideation phase and gave me new ways to think about diverse user needs. However, I encountered limitations such as generic or biased content, which required multiple iterations and critical engagement. Through this process, I learned to refine prompts and interpret responses rather than accept outputs at face value. Next time, I will plan more time for prompt testing and explore multiple tools to triangulate better insights. Reviewing and revising AI outputs improved my confidence in both content generation and ethical web design.

Integrity Statement

I used GenAI tools as a supplement to my own design and research efforts. All AI-generated content was carefully reviewed, edited, and integrated to align with academic expectations. This acknowledgement ensures transparency and supports my learning, while maintaining the integrity of this assessment task.
