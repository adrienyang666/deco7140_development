GenAI & Machine Translation Acknowledgement

This page documents my use of Generative AI (GenAI) and Machine Translation (MT) tools in the development of the CCI Community Hub project, in line with UQ's GenAI/MT Usage Framework and course requirements.

Reflection on GenAI/MT Use

  1. Why did I use GenAI or MT tools?
    I used GenAI (such as ChatGPT and Copilot) primarily to generate creative ideas, rephrase English instructions, draft sample code, and help with bilingual communication (English–Chinese) for clarity and accessibility. These tools enhanced my productivity and creativity, especially for brainstorming and refining user-facing content.
  2. How did I ensure academic integrity?
    I critically reviewed, edited, and verified all GenAI/MT outputs. I ensured that no assessment answers, code, or design elements were directly copied without modification. All AI-generated content was validated against course requirements, ensuring originality and alignment with assignment instructions.
  3. Ethical and responsible use:
    I used GenAI/MT as a support tool rather than a replacement for my own critical thinking. All content and code were subject to my own quality assurance. I avoided using GenAI/MT for any restricted tasks (e.g., direct plagiarism or assessment loopholes), following both UQ and course guidelines.

GenAI & MT Tools Used

Tool How It Was Used Where in the Project
ChatGPT (OpenAI) Brainstorming design ideas, drafting/refining English text, generating sample code and accessibility suggestions, bilingual translation and proofreading All pages (concepts, text drafts, accessibility)
Copilot (GitHub) Code auto-completion and structure suggestions JavaScript modules, CSS organization
DeepL/Google Translate Initial machine translation of text (followed by manual editing for clarity and accuracy) Bilingual explanations and page drafts

Integrity Statement

I confirm that I have used GenAI and MT tools in accordance with UQ’s GenAI and MT Usage Framework. All AI/MT outputs have been critically reviewed, substantially modified, and integrated into my own work. No GenAI/MT outputs have been submitted as-is for assessment. The intellectual work, design, and academic integrity of this submission are my own responsibility.
