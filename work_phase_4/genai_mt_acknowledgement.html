GenAI & Machine Translation Acknowledgement

This page is required by the DECO7140 assessment to transparently document my use of Generative AI (GenAI) and Machine Translation (MT) tools, in line with UQ's GenAI/MT Usage Framework and course requirements. All major AI/MT use is declared below.

Reflection on GenAI/MT Use

  1. Why did I use GenAI or MT tools?
    I used GenAI (such as ChatGPT and Copilot) to enhance productivity, brainstorm creative ideas, draft and refine English instructions, generate sample code, and support bilingual communication (English–Chinese) for clarity and accessibility. These tools were especially helpful for ideation, language clarity, and code prototyping.
  2. How did I ensure academic integrity?
    I critically reviewed, edited, and verified all GenAI/MT outputs. No assessment answer, code, or design was directly copied; all AI-generated content was carefully validated and substantially rewritten to match assignment requirements and maintain originality.
  3. Ethical and responsible use:
    I used GenAI/MT strictly as a support tool, not a replacement for my own thinking. All code and content were subject to my own quality control. I avoided using GenAI/MT for restricted or prohibited tasks, following both UQ and course policies.

GenAI & MT Tools Used

Tool How It Was Used Where in the Project
ChatGPT (OpenAI) Brainstorming design ideas, drafting/refining English text, generating sample code and accessibility suggestions, bilingual translation and proofreading Content and code drafts, accessibility and language checks on all pages
Copilot (GitHub) Code auto-completion and structure suggestions for HTML, CSS, JS JavaScript modules, CSS/HTML refactoring
DeepL / Google Translate Initial machine translation for bilingual content, always followed by manual editing for clarity and accuracy Bilingual explanations, page drafts, project documentation
DALL·E (OpenAI) / Other Image AI Generated comic-style illustrations for community events, team, and stories; all images reviewed and described with alt text for accessibility Homepage highlight cards, team poster, directory/events images, story illustrations

All images on this website were generated using GenAI tools and have been reviewed for appropriateness, accessibility, and inclusivity.

Integrity Statement

I confirm that I have used GenAI and MT tools in accordance with UQ’s GenAI and MT Usage Framework. All AI/MT outputs have been critically reviewed, substantially modified, and integrated into my own work. No GenAI/MT outputs have been submitted as-is for assessment. The intellectual work, design, and academic integrity of this submission are my own responsibility.
